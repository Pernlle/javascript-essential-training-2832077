Inline JavaScript

You can write JavaScript directly inside an HTML document using the script element. The element can be added in the head section or inside the body element.

-->Script text er til for at indkoorporere JS in line. Browseren stopper ALT rendering mens den gennemgår et script. Dette er fordi at et script muligvis ændre elementer i siden. Denne form for JS findes kun på denne side, og effekterer ikke andre dokumenter. Det er muligt at have sine styles i et andet sheet, dette kan også gøres for script filer som et script.js, som kan referes tilog gennerellisere gennem alle sider
